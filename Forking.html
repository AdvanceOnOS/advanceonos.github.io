Forking Guide

Visit Our Github Repository

Click the "Code" button and select "Download ZIP"

Once downloaded, extract the ZIP file

Customize

Like a wallpaper? get your image file, then copy your image to "universal/../System/Scripts/Wallpaper/Assets".

For a different name or file extension, open up "universal/../System/Scripts/Wallpaper/Wallpaper.css" in your preferred code editor and edit "background-image: url("Assets/default.jpg");" to "background-image: url("Assets/example.png");", "example.png" being replaced with your image name and extension

Additional Info

In the universal folder ".." is replaced with either "Desktop" or "Mobile"
